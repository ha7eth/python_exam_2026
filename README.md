## Задание

1. Работа выполняется в **индивидуальной ветке**. Название ветки: `имя_фамилия_final` (например, `ivan_ivanov_final`).
2. В корне репозитория создайте папку со своим именем (например, `ivanov_work/`).
3. Все решение должно быть оформлено в виде одного файла **Jupyter Notebook (`.ipynb`)**.
4. После завершения работы необходимо создать **Pull Request** из вашей ветки в ветку `main`.

---

Ваш Jupyter Notebook должен содержать следующие разделы:

1.  **Описание (Markdown):** Номер билета, название формулы и её запись в формате LaTeX.
2.  **Загрузка данных:** Чтение чисел из вашего текстового файла (каждое число — это координата `x`). Используйте относительный путь `../data/...`.
3.  **Обработка:** Реализация функции(ий) для расчета значений `y` на основе прочитанных данных. Используйте модуль `math`.
4.  **Визуализация:** Построение графика с использованием `matplotlib`.
    * Добавьте сетку (`grid`), подписи осей и легенду.
    * Выберите цвет линии, отличный от стандартного.
    * Сохраните итоговый график в формате `.png` прямо внутри ноутбука.
5.  **Вывод:** Краткое текстовое описание полученного результата (1-2 предложения).
